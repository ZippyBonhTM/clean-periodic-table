# Feature: Periodic Table

### Acessar a Tabela Periódica
>Dado que o usuário acesse a página da Tabela Periódica<br>
A Tabela Periódica deve estar visível com todos os Elementos Químicos.

### Interagir com Elemento Químicos
> Dado que o usuário acesse a Tabela Periódica<br>
E clique em um Elemento Químico<br>
Deve ser mostrado uma tabela contendo todas as propriedades do Elemento Químico.
